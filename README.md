# DayZero Intelligence

> **An intelligent analytics platform that delivers value before data ingestion**

[![View Live Demo](https://img.shields.io/badge/View-Live%20Demo-0066ff?style=for-the-badge)](https://alonhaze7.github.io/DayZero/)

## 🚀 Overview

DayZero Intelligence solves the cold-start problem that kills most analytics platforms. By combining qualitative brand intelligence with immediate quantitative insights from paid media APIs, it delivers actionable decisions within **hours**, not quarters.

## ✨ Key Features

- **⚡ Day-Zero Value** — Intelligence before data ingestion
- **🎯 Action Layer** — One-click decisions with confidence scores
- **🔄 Self-Healing** — Automated fixes for config gaps
- **🔍 Platform Expertise** — Different failure modes for Google vs Meta
- **📊 Auditable Intelligence** — Every insight traceable to specific API fields
- **🎨 Brand-Conditioned** — Mission and positioning inform all recommendations
- **🔎 Competitive Intelligence** — Public ad data analyzed before client connects anything

## 🎯 The Problem

Most analytics products need historical data to accumulate before providing value. Clients wait weeks or months before seeing their first actionable insight, creating a "valley of zero value."

## 💡 The Solution

### Three-Phase Approach

#### 1. Pre-Connect Phase (T-0)
**Brand Intelligence First**

Client provides:
- Website URL
- 3-5 competitor names
- Optional mission statement

We deliver:
- **AI-drafted brand brief** (mission, tone, ICP, differentiation)
- **Competitor ad teardown** from Meta Ad Library + Google Transparency Center
- **White-space analysis** showing positioning opportunities

#### 2. Day-Zero Phase (T+00:00 to T+01:40)
**Quantitative Intelligence**

1. Connect Google Ads and Meta accounts
2. Backfill 30 days of history via APIs (automatic)
3. Audit surfaces 10 platform-specific findings
4. 6 one-click decisions with confidence scores

**Value delivered in under 2 hours.**

#### 3. Day 30+ Phase
**Compounding Intelligence**

- Trend detection and forecasting
- Paid-plus-owned media attribution
- LTV-based campaign performance
- Creative resonance across channels

## 🏗️ Technical Architecture

### Data Sources

#### Paid Media APIs
- **Google Ads API** (v14+) — 30-90 day backfill
- **Meta Marketing API** (v18+) — 30 day backfill
- **DV360 API** — 30 day backfill
- **TikTok Marketing API** — 30 day backfill

#### Public Competitive Data
- **Meta Ad Library API** — Live competitor ads
- **Google Ads Transparency Center** — Competitor creative (requires scraping)

#### Brand Intelligence
- **Website scraping** — Mission, positioning, messaging
- **LLM inference** — Ad angle classification

### Self-Healing Actions

Two automated fixes that mirror existing pipeline-health loops:

1. **Fix Tracking** — Detect conversion action gaps, create and link actions automatically
2. **Turn on CAPI** — Configure Conversions API endpoint, map events, validate
